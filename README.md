# 📌 QR-Based Attendance System

A modern and responsive **QR-Based Attendance Management System** developed using **React.js**. This project provides an easy way to manage students, faculty, subjects, attendance records, reports, and settings through a user-friendly admin dashboard.

---

## 🚀 Features

- 🔐 Admin Login Authentication
- 📊 Dashboard with Summary Cards
- 👨‍🎓 Student Management
  - View Students
  - Edit Students
  - Delete Students
  - Student Details
- 👨‍🏫 Faculty Management
  - Faculty List
  - Faculty Details
- 📚 Subject Management
  - Subject List
  - Subject Details
- ✅ Attendance Management
  - Attendance Summary Cards
  - Search & Filter
  - Present/Absent Status
- 📈 Reports Dashboard
- ⚙️ Settings Page
- 💾 Local Storage Support
- 🚪 Logout Functionality
- 📱 Responsive UI Design
